## Project Overview

This repository is for `nexus-factory`.

## Agent Guidelines

- Keep changes focused and easy to review.
- Prefer simple, readable implementations before adding abstractions.
- Update documentation when behavior or setup steps change.
- Add or update tests when making behavior changes.
- Do not commit secrets, credentials, or local machine-specific files.

## Verification

Before finishing a change, run the most relevant checks available in the repository and summarize what passed or could not be run.
